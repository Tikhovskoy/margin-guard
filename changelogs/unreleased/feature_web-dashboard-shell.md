## [MR](https://github.com/Tikhovskoy/margin-guard/pull/0)

### Added

- Модульный frontend dashboard в `apps/web` с таблицей маржи, alert-панелью и настройкой порога.
- Интерактивный поиск SKU и фильтр проблемных товаров.
- Базовый API-клиент и конфигурация URL backend для последующего подключения live-данных.
- Получение preview маржи из FastAPI и загрузка CSV себестоимости прямо из dashboard.

- Полностью переработан интерфейс dashboard: добавлены премиальная визуальная система, плавные анимации и адаптивная компоновка.
- Экран разделён на независимые компоненты обзора, навигации и таблицы товаров для дальнейшего масштабирования.
- CI дополнен проверками lint, тестов и production-сборки web dashboard.
